Command Explanation User Level
!addcom {name} {response} Adds a command to the bot, {name} is what's used to do the command and response is what the bot responds with. Mod+
!delcom {name} Deletes a command from the bot, {name} is the name of the command you want to delete! Mod+
!urban {term} Searches UrbanDictionary for {term} and responds with the top result. Mod+
!addquote {quote} Adds a quote to the bot, {quote} is what the quote says. The bot responds with the ID. Mod+
!delquote {id} Removes a quote from the bot, {id} is the ID of the quote you wanna remove. Mod+
!bug Gives ways to report bugs to me! Mod+
!quote {id} Get a quote from the bot, {id} is the ID of the quote. Anyone
!ping Check to see if the bot is lagging or not! Anyone
